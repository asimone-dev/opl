# Organization of Programming Languages (OPL)

This repository supports coursework and project development for the OPL course.
This repository is organized as follows:

## Repository Structure / Folder Descriptions

- **notes/**
    Lecture notes and written summaries related to programming language concepts, paradigms, and theory.

- **projects/**
    Course projects, each organized to support active development, weekly progress submissions, and final Canvas submissions.

- **templates/**
    Reusable templates for required projects and documentation sush as rubrics, activity logs, and technical manuals.
